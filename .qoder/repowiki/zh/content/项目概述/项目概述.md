## 引言
FastapiAdmin 是一款基于 FastAPI + Vue3 + TypeScript 的全栈快速开发平台，专为企业级中后台系统的快速搭建而设计。项目采用"开箱即用"的理念，通过标准化的工程结构和丰富的内置功能，让开发者能够在5分钟内完成从零到一的项目搭建。

## 快速开始
5分钟本地运行指南：

```bash
# 1. 克隆项目
git clone https://gitee.com/fastapiadmin/FastapiAdmin.git

# 2. 配置环境变量
cp backend/env/.env.dev.example backend/env/.env.dev
cp frontend/web/.env.development.example frontend/web/.env.development

# 3. 启动后端服务（自动建表 + 初始化数据）
cd backend && uv sync && uv run main.py run --env=dev

# 4. 启动前端服务
cd ../frontend/web && pnpm install && pnpm run dev

# ✅ 浏览器打开 http://127.0.0.1:5173，使用 admin/123456 登录
```

**环境要求**
- Python ≥ 3.10（推荐 3.12）
- Node.js ≥ 20.0 + pnpm
- MySQL 8.0+ / PostgreSQL 14+
- Redis 6.x / 7.x

## 工程结构
FastapiAdmin 采用 Monorepo 全栈工程结构，包含完整的前后端和文档体系：

```
FastapiAdmin/            # 主项目目录
├─ backend/              # FastAPI 后端（Pydantic 2.0 + SQLAlchemy + Alembic）
├─ frontend/             # 前端工程集合
│   ├── web/             # Vue3 Web 前端（Element Plus + TypeScript）
│   ├── app/             # UniApp 移动端（H5 + 小程序 + App）
│   └── docs/            # VitePress 文档网站
├─ docker/               # Docker Compose 一键部署（Nginx + SSL）
├─ deploy.sh             # 一键部署脚本
└─ LICENSE               # MIT 开源协议
```

## 技术栈概览
### 后端技术栈
- **核心框架**：FastAPI 0.115.2 + Pydantic 2.5.2 + SQLAlchemy 2.0.45
- **数据库**：MySQL 8.0+ / PostgreSQL 14+ / SQLite（异步驱动）
- **缓存**：Redis 7.1.0 + FastAPILimiter（请求限流）
- **任务调度**：APScheduler 3.11.0 + Prefect 3.5.0
- **部署**：Uvicorn 0.30.6 + Gunicorn + Docker

### 前端技术栈
- **核心框架**：Vue3 3.5.34 + TypeScript + Vite 7.1.5
- **UI 组件**：Element Plus 2.13.7 + Wot Design Uni
- **状态管理**：Pinia 3.0.4 + Vue Router 5.0.7
- **可视化**：ECharts 6.0.0 + Vue Flow
- **开发工具**：Axios 1.16.1 + ExcelJS 4.4.0
